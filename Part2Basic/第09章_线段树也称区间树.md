# 第09章 线段树
> 也称区间树，表明我们关系的是一个区间内的问题
## 9.1 什么是线段树
> 对于有一类问题，我们关系地是线段(即区间)

### 线段树的应用举例
+ 染色问题
  > ![区间染色问题1](images/第09章_区间树/区间染色问题1.png)
  > ![区间染色问题1](images/第09章_区间树/区间染色问题2.png)
  
  > 染色问题的操作步骤如下：

  | 操作               | 使用数组实现 | 使用线段树实现 |
  | ------------------ | ------------ | -------------- |
  | 染色操作(更新区间) | O(n)         | O(logn)        |
  | 查询操作(查询区间) | O(n)         | O(logn)        |
